# 📊 Data Analyst Projects

A collection of Python data analysis projects using NumPy, Pandas, and Matplotlib.

## 📌 About

This repository contains several beginner to intermediate data analysis projects created to practice data manipulation, statistical analysis, and data visualization with Python.

## 🛠 Technologies

- Python 3
- NumPy
- Pandas
- Matplotlib

## 📂 Projects

- 📈 Monthly Sales Analysis
- 🌡 Temperature Changes During the Week
- 🎓 Student Grades Analysis
- 📚 Study Hours vs Student Scores
- 🏫 Classroom Information Management
- 📝 Student Grade Registration System

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/arshiasadri/data-analyst.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run a project:

```bash
python monthly_sales_volum.py
```
